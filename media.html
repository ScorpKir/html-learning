Медиаконтент

В понятие "медиаконтент" входит:

  • Аудио
  • Видео
  • Картинки
  • Фреймы

Видео

Тег <video> отображает видеоплеер на странице.

Можно добавить элементы управления плеером видео с помощью атрибута controls.

Задать автовоспроизведение видео при загрузке страницы можно с помощью атрибута autoplay.

Автовоспроизведение во многих браузерах работает только совместно с атрибутом muted, который ставит видео в беззвучный режим.

Чтобы видео начинало воспроизведение заново после его завершения нужно задать тегу <video> атрибут loop

Атрибут poster задаёт изображение как превью для видео. Без этого атрибута в качестве превью у видео будет первый кадр.

Пример:

        
            <video width="500" height="300" poster="/images/ass-penetration.jpg" controls>
                <source src="/videos/6548176-hd_1920_1080_24fps.mp4" />
            </video>

Аудио

Тег <audio> отображает видеоплеер на странице.

Атрибуты все такие-же как и у видеоплеера.

Пример

        
            <audio controls>
                <source src="/audio/salam-alekum.mp3.mp3" />
            </audio>

Фреймы

Тег <iframe> отображает виджет на странице. Может использоваться для отображения виджетов с погодой, карт и тд.

Пример

        
            <iframe
              src="https://yandex.ru/map-widget/v1/?um=constructor%3A65c8383aa944e7309ececed86f938ff496e70866a6a44215d87052b39a9ec002&amp;source=constructor"
              width="500"
              height="400"
            >
            </iframe>
